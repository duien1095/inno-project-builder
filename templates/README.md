# Templates

Thư mục này chứa các template mẫu dùng để khởi tạo cấu trúc dự án.

## Template hiện có
- QHC
- QHCT
- TOD

## Cách thêm template mới
1. Tạo một thư mục mới trong thư mục templates/
2. Đặt cấu trúc thư mục và file cần thiết vào thư mục mới đó
3. Khởi động lại ứng dụng, template sẽ tự xuất hiện

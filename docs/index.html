INNO PROJECT BUILDER

Khởi tạo cấu trúc dự án quy hoạch chuẩn — Portable & Miễn phí

🚀 Tải ngay phiên bản mới nhất

Ứng dụng desktop giúp khởi tạo nhanh cấu trúc thư mục dự án quy hoạch chỉ với một cú nhấp chuột. Không cần cài đặt Python.

⬇ Tải về cho Windows project_init_portable.exe
Đang tải thông tin phiên bản...

📋 Tính năng

  • Tạo cấu trúc thư mục theo template chuẩn
  • Hỗ trợ 5 loại đồ án quy hoạch
  • Giao diện trực quan, dễ sử dụng
  • Kiểm tra an toàn đường dẫn
  • Chạy portable, không cần cài đặt

📁 Template có sẵn

Ứng dụng hỗ trợ các template sau:

QHC QHPK QHCT CONCEPT_QHPK CONCEPT_QHCT

Mỗi template tự động tạo cấu trúc thư mục INPUT → PROCESS → OUTPUT phù hợp với từng loại đồ án.

🔧 Yêu cầu hệ thống

  • Windows 7 trở lên (64-bit)
  • Dung lượng ổ đĩa trống: ~50 MB
  • RAM: tối thiểu 256 MB
  • Không cần cài Python

📖 Hướng dẫn sử dụng

  • Tải file project_init_portable.exe
  • Chạy file .exe (có thể bị Windows SmartScreen chặn, nhấn "Run anyway")
  • Chọn thư mục dự án → Chọn template → Nhấn KHỞI TẠO
  • Cấu trúc thư mục sẽ được tạo tự động

🛠 Dành cho nhà phát triển

Build portable từ source code: 

git clone https://github.com/duien1095/inno-project-builder.git
cd inno-project-builder
pip install -r requirements-release.txt
release.bat 0.1.0

Kết quả: dist/project_init_portable.exe
